# Memorizing Game

> ## 使用語言

1. HTML
2. Sass/SCSS
3. JavaScript--MVC 架構

> ## 使用說明

### 1. Game Rule

1. 點擊 Let's play！開始遊戲

### 2. Play Game

1. 右上的回合數+1
2. 點擊卡片進行翻牌
3. 不論花色，找出牌面數字相同的兩張卡片及配對成功
4. 配對失敗的兩張卡片會閃爍 1 秒鐘的時間供您記憶卡片位置
5. 每配對成功一次，右上的嘗試次數+1 且總分加 10 分，直到總分為 260 分即過關

### 3. Game Ending

1. 點擊 Let's play again！可再次開始遊戲
